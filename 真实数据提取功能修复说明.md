## 🔄 数据来源

所有数据都来自 DealerDashboard 组件的：

1. **getSeries 方法**：
   ```javascript
   getSeries(key) {
     return this.months.map((m) => 
       this.toNumber(this.currentDealer[metricKeys[key](m)])
     )
   }
   ```

2. **currentDealer computed 属性**：
   ```javascript
   currentDealer() {
     return this.dealers.find((d) => 
       d['经销商代码'] === this.selectedCode
     ) || {}
   }
   ```

3. **monthSnapshots computed 属性**：
   ```javascript
   monthSnapshots() {
     return this.months.map((m) => ({
       month: m,
       sales: this.toNumber(this.currentDealer[metricKeys.销量(m)]),
       traffic: this.toNumber(this.currentDealer[metricKeys.客流量(m)]),
       ...
     }))
   }
   ```

## ✨ 关键改进

### 之前（写死的数据）
```javascript
function extractTrendData(vm) {
  return {
    months: ['1月', '2月', ...],
    sales: [120, 132, 101, ...],  // 写死的！
    leads: [220, 182, 191, ...],  // 写死的！
    potential: [150, 232, 201, ...]  // 写死的！
  };
}
```

### 现在（真实数据）
```javascript
function extractTrendData(vm) {
  const months = vm.filteredMonths || vm.months || [];
  const salesData = vm.getSeries ? vm.getSeries('销量') : [];  // 真实数据！
  const leadsData = vm.getSeries ? vm.getSeries('线索量') : [];  // 真实数据！
  const potentialData = vm.getSeries ? vm.getSeries('潜客量') : [];  // 真实数据！
  
  return {
    months: months,
    sales: salesData,
    leads: leadsData,
    potential: potentialData
  };
}
```

## 📌 注意事项

1. **数据依赖 getSeries 方法**：
   - 如果 vm.getSeries 不存在，会返回空数组
   - 确保 DealerDashboard 组件有 getSeries 方法

2. **数据格式**：
   - 所有数值都经过 toNumber 处理
   - 百分比数据会自动转换（如成交率）
   - 计算结果会四舍五入

3. **时间范围**：
   - 使用 vm.filteredMonths 支持时间范围过滤
   - 支持全部时间/按季度/按月度
