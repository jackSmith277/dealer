## 🎨 统一的样式特点

### 1. 简洁的配置
- 只保留必要的属性
- 不自定义图标大小和形状
- 使用默认的间距和宽度

### 2. 一致的文本样式
```javascript
textStyle: { color: '#6b7280' }
```
- 颜色：#6b7280（灰色）
- 字体大小：默认（不指定）
- 字体粗细：默认（不指定）

### 3. 统一的布局
```javascript
top: 10,           // 距离顶部10px
left: 'center',    // 水平居中
orient: 'horizontal'  // 水平排列
```

### 4. 简单的数据格式
```javascript
data: ['系列1', '系列2', '系列3']  // 字符串数组
```
而不是：
```javascript
data: [
  { name: '系列1', icon: 'rect' },  // ❌ 对象数组
  { name: '系列2', icon: 'rect' }
]
```
