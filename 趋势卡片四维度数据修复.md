# 趋势卡片四维度数据修复

## ✅ 问题已解决

### 问题描述
第一张卡片（销量/线索/潜客月度走势）应该包含四个维度的数据：
1. 销量
2. 客流量
3. 线索量
4. 潜客量

但系统只提取了三个维度（缺少客流量）。

### 修复内容

#### 1. dataExtractor.js - extractTrendData 函数

**修复前**（只有3个维度）：
```javascript
function extractTrendData(vm) {
  const salesData = vm.getSeries ? vm.getSeries('销量') : [];
  const leadsData = vm.getSeries ? vm.getSeries('线索量') : [];
  const potentialData = vm.getSeries ? vm.getSeries('潜客量') : [];
  
  return {
    months: months,
    sales: salesData,
    leads: leadsData,
    potential: potentialData
  };
}
```

**修复后**（4个维度）：
```javascript
function extractTrendData(vm) {
  const salesData = vm.getSeries ? vm.getSeries('销量') : [];
  const trafficData = vm.getSeries ? vm.getSeries('客流量') : [];  // ✅ 新增
  const leadsData = vm.getSeries ? vm.getSeries('线索量') : [];
  const potentialData = vm.getSeries ? vm.getSeries('潜客量') : [];
  
  console.log('提取趋势数据 - 客流量:', trafficData);  // ✅ 新增日志
  
  return {
    months: months,
    sales: salesData,
    traffic: trafficData,  // ✅ 新增
    leads: leadsData,
    potential: potentialData
  };
}
```
